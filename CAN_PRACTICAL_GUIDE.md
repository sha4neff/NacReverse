# PSA NAC CAN — Полный практический справочник

**Что это:** Справочник для работы с CAN шиной магнитолы NAC (Peugeot/Citroen/DS/Opel)
**Как пользоваться:** Найди нужный CAN ID → посмотри что делает каждый байт → отправь/прими данные
**Всего:** 99 CAN фреймов, 927 сигналов

---

## Направления

- **RX (от авто к магнитоле):** Магнитола ПРИНИМАЕТ эти фреймы от BSI/других блоков
- **TX (от магнитолы к авто):** Магнитола ОТПРАВЛЯЕТ эти фреймы (управление)

## 0x350 — Состояние климата (RX, 8 байт)

Магнитола ПОЛУЧАЕТ это от блока климата.

| Байт | Бит | Что это | Значения |
|------|-----|---------|----------|
| 0 | 0 | Кондиционер AC | 0=выкл, 1=вкл |
| 0 | 1-2 | Режим работы | 0=выкл, 1=ручной, 2=авто |
| 0 | 3-4 | Режим рециркуляции | 0=выкл, 1=вкл, 2=авто |
| 0 | 5 | Статус системы готов | 0=нет, 1=да |
| 0 | 6 | Mono-zone активен | 0=нет, 1=да |
| 0 | 7-1.3 | Скорость вентилятора | 0-8 (0=выкл, 8=макс/авто) |
| 1 | 4-5 | Направление обдува | 0=лицо, 1=лицо+ноги, 2=ноги, 3=стекло |
| 1 | 6-2.2 | Направление расширенное | 5 бит, комбинации |
| 2 | 3 | AQS активен | 0=нет, 1=да |
| 2 | 4-6 | Уровень AQS | 0-7 |
| 2 | 7-3.2 | Состояние AC | комплексное |
| 3 | 3-6 | Резерв | — |
| 4 | 0-3 | Температура левая (low) | см. формулу |
| 4 | 4 | Обогрев стекла | 0=выкл, 1=вкл |
| 4 | 5-6 | Статус обогрева | 0=выкл, 1=ожидание, 2=работает |
| 4 | 7-5.1 | Темп. правая (мл. часть) | см. формулу |
| 5 | 1-2 | Резерв | — |

**Формула температуры:** `raw / 2 + 14 = °C` (диапазон 14-30°C)

**Пример:** `350: 03 25 08 00 3C 00 00 00`
- AC=вкл, AUTO=вкл, вентилятор=5, температура=22°C

# СКОРОСТЬ, ОБОРОТЫ, ДВИЖЕНИЕ

## 0x036 — Скорость автомобиля (RX, 8 байт)

| Байт | Бит | Что это | Формула |
|------|-----|---------|---------|
| 0 | 0-7 | Скорость low byte | (byte0 + byte1*256) / 100 = км/ч |
| 1 | 0-7 | Скорость high byte | |
| 2 | 0-7 | Резерв | — |
| 3 | 0-7 | Резерв | — |
| 4-7 | 0-7 | Одометр (32 бит) | км (little-endian) |

**Пример:** `036: 88 13 00 00 40 42 0F 00`
- Скорость: 0x1388 = 5000 → 50.00 км/ч
- Одометр: 0x000F4240 = 1000000 → 10000.00 км

---

## 0x069 — Обороты двигателя (RX, 4 байта)

| Байт | Бит | Что это | Формула |
|------|-----|---------|---------|
| 0 | 0-7 | Обороты low byte | (byte0 + byte1*256) * 0.125 = RPM |
| 1 | 0-7 | Обороты high byte | |
| 2 | 0-7 | Резерв | — |
| 3 | 0-7 | Резерв | — |

**Пример:** `069: 00 10 00 00`
- Обороты: 0x1000 = 4096 → 4096 * 0.125 = 512 RPM (холостой ход)

**Пример:** `069: 00 40 00 00`
- Обороты: 0x4000 = 16384 → 16384 * 0.125 = 2048 RPM

---

## 0x128 — Угловая скорость / Ускорение (RX, 4 байта)

| Байт | Бит | Что это | Формула |

| 2-3 | 0-15 | Поперечное ускорение | (raw - 32768) / 100 = м/с² |

**Пример:** Прямо = 0x8000 (32768) → 0 °/с

# ВРЕМЯ И ДАТА

## 0x01C — Время от BSI (RX, 4 байта)

| Байт | Бит | Что это | Значения |
|------|-----|---------|----------|
| 0 | 0-5 | Секунды | 0-59 |
| 0 | 6-7 + 1.0-3 | Минуты | 0-59 (6 бит, разбито) |
| 1 | 4-7 + 2.0 | Часы | 0-23 (5 бит, разбито) |
| 2 | 1-5 | День | 1-31 |
| 2 | 6-7 + 3.0-1 | Месяц | 1-12 (4 бит, разбито) |
| 3 | 2-7 | Год | +2000 (пример: 24=2024) |

**Декодирование:**
```
секунды = byte0 & 0x3F
минуты = ((byte0 >> 6) & 0x03) | ((byte1 & 0x0F) << 2)
часы = ((byte1 >> 4) & 0x0F) | ((byte2 & 0x01) << 4)
день = (byte2 >> 1) & 0x1F
месяц = ((byte2 >> 6) & 0x03) | ((byte3 & 0x03) << 2)
год = ((byte3 >> 2) & 0x3F) + 2000
```

# ФОРМУЛЫ ПРЕОБРАЗОВАНИЯ

| Параметр | Формула | Пример |
|----------|---------|--------|
| Температура климата | raw / 2 + 14 = °C | 16 → 22°C |
| Температура общая | raw - 40 = °C | 62 → 22°C |
| Скорость | (b0 + b1×256) / 100 | 0x1388 → 50 км/ч |
| Обороты | (b0 + b1×256) × 0.125 | 0x1000 → 512 RPM |
| Расход топлива | raw / 10 = л/100км | 80 → 8.0 |
| Давление шин | raw × 0.025 = бар | 92 → 2.3 бар |
| Угол руля | (raw - 8192) / 10 = градусы | 8692 → +50° |
| Напряжение | raw / 10 = В | 142 → 14.2В |

# ПРАКТИЧЕСКИЕ ПРИМЕРЫ

## Включить кондиционер

```
TX 1B6: 01 00 00 00 00 00 00 00
        │
        └── бит 0 = 1 (AC вкл)
```

## Установить температуру 24°C

```
TX 1B6: 00 00 14 14 00 00 00 00
              │  │
              │  └── правая: (24-14)*2 = 20 = 0x14
              └───── левая: (24-14)*2 = 20 = 0x14
```

## Включить LED парктроника

```
TX 2A2: 01 00 00 00 00 00 00 00
        │
        └── биты 0-1 = 01 (LED вкл)
```

## Мигать LED ESP

```
TX 2A2: C0 00 00 00 00 00 00 00
        │
        └── биты 6-7 = 10 (LED ESP мигает)
```

## Читать скорость из 0x036

```python
# Получили: 036: 88 13 00 00 00 00 00 00
raw = 0x1388  # byte0 + byte1*256
speed_kmh = raw / 100  # = 50.00 км/ч
```

## Читать температуру из 0x112

```python
# Получили: 112: 7C 7C 50 00 00 00
raw_driver = 0x7C  # = 124
temp_c = (raw_driver / 2) - 40  # = 62 - 40 = 22°C
```
